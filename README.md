## Setup & Run

```bash
# 1. Install dependencies
pnpm install

# 2. Start development server
pnpm dev

# 3. Open in browser
# http://localhost:5173
```

## Pages

| Route | Page |
|-------|------|
| `/login` | Login |
| `/register` | Register |
| `/verify-email` | OTP Verify |
| `/dashboard` | Dashboard |
